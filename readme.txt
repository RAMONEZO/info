Клонирование проекта в папку nameProject:
git clone https://... nameProject

Добавление файлов в коммит:
git add . 

Запись коммита:
git commit -m "Комментарий коммита"

Отправить коммит:
git push

Принять изменения:
git pull

Авторство коммитов:
git --global user.email "Ваш емайл"
git --global user.name "Ваше имяя"
